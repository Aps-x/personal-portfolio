NOTICE: Unity HTML5 WebGL builds run poorly on Firefox; additionally, entering and then exiting fullscreen can cause visual glitches on Firefox. Chrome is recommended.

Description

This was my second game programmed in Unity. This project took 10 days to complete (21/01/2022 - 30/01/2022). It is a clone of the 1978 classic arcade title "Space Invaders" by Taito, or more specifically, a clone of the later 1985 remake for the SG-1000 console by SEGA. This clone of Space Invaders is meant as a learning exercise / for educational purposes.
