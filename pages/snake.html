Description

This was my third game programmed in Unity. This was originally intended to be an interim project but I ended up fully polishing it. This project took 6 days to complete at a more relaxed pace (3/02/2022 - 8/02/2022). It is a clone of the classic game "Snake", which has its roots in the 1976 arcade game "Blockade". This version copies the aesthetic of "Snake II" by Nokia and the gameplay settings/modes from Google Snake. This clone of Snake is meant as a learning exercise / for educational purposes.
