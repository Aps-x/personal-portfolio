NOTICE: Originally designed for desktop not for websites. Chrome is recommended.

Controls

[A] or [←] - Rotate left

[D] or [→] - Rotate right

[Space] - Shoot

Description

This was my first game programmed in Unity. This project took 5 days to complete (14/01/2022 - 18/01/2022). It is a clone of the classic arcade title "Asteroids", originally released in 1979 by Atari. This clone of Asteroids is meant as a learning exercise / for educational purposes. All rights belong to the current IP owners of Asteroids.

This game was designed for, and is best played on, desktop.
